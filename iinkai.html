大学祭実行委員会

大学祭実行委員会は9個の局に分かれて仕事を分担し、中心となって大学祭を盛り上げる活動をしています。
ここではそれぞれの局がどのような仕事をしているか紹介しています。
学年、時期問わず興味のある方はいつでも大歓迎！学生会館二階に大祭室があるので、気軽にお立ち寄りください！

委員長局 総務局 システム局 会計局 企画局 模擬局 装飾局 広報局 カフェ局

    委員長局

    　メンバーは、3年生と2年生ともに3人ずつの計6人！学年や性別の壁を感じない雰囲気が最高です！マックのポテトを与えておけばおとなしくなります(写真見てね笑) お仕事としては、大学祭全体のまとめ役です。といっても堅苦しいお偉いさんではなく、各局のお手伝いや相談役、そしてイベントの企画などをやっています！ そして委員長局に入ったからといって来年必ず委員長にならないといけない、というのは全くありません！ご安心を～
    いろんなことをやってみたい！大学の雰囲気を知りたい！おいしいご飯をみんなと食べたい！ポテト大好き！というそこのあなた、是非委員長局へ！

    -------- 仕事内容 --------

  • 大学祭全体をまとめる局
  • 仕事の振り分けと大学祭全体の管理
  • 各局のサポートに回る
  • バーベキューなどのイベントの企画

    • -------- メンバー --------

  • 委員長：三上　航平
  • 副委員長：藤本　幸　　
  • 副委員長：岩城　のどか

    総務局

    　総務局は基本的に仕事数が少なく、空き時間には喋ったりトランプしたりなどして交友を深めることを出来ます。 また、総務局自体の仕事が少ない分他の局に手伝いに行くことができ、多くの人と関わりを持ちたい人などにとって非常に都合のいい局だと言えます！
    仕事内容は机、椅子等の管理や土嚢作成などの単純作業が多いです。また、総務局は男女比1:1なので、男子学生も女子学生も入りやすい環境になっています！ 遠慮なく声かけてもらって大丈夫なので、気軽にお声かけください！

    -------- 仕事内容 --------

  • 備品の管理
  • 備品の貸し出し
  • 備品の点検

    • -------- メンバー --------

  • 局長：小林　雅明

    システム局

    　今年新設のシステム局です！仕事内容は出店者とのやり取りをするサイトの管理やホームページ更新がメインになります。 3年生3人、2年生4人(男子3人女子4人、みんな情報)で活動中！全員兼部しています！ちなみに局長(佐藤)は吹奏楽部です！
    Webアプリやプログラミングに興味がある人、ちょっとでもやってみたいな～と思ったらぜひシステム局へ！

    -------- 仕事内容 --------

  • サイトの管理
  • ホームページの更新

    • -------- メンバー --------

  • 局長：佐藤　優紀
  • 副局長：兼定　諒
  • 副局長：上田　徳彦

    模擬局

    　私たち模擬局は3年生5人、2年生3人で活動しています。主な活動内容は模擬店(出店)に関するお仕事です。 模擬店の出店者と備品を管理する業者様の仲介を行います。他にも模擬店のテント設営や、模擬店の出店もしたりします。 テント設営は他の局と合同で行うので心配無用！模擬店も皆さんのアイディアをたくさん取り入れてたいと思っています！
    新入生から3年生まで壁を作らないことを目標に活動していきます！ラフな気持ちでゆっくり楽しんでいきましょう！

    -------- 仕事内容 --------

  • 出店者会議
  • 業者様との連携
  • テント設営
  • 模擬店出店

    • -------- メンバー --------

  • 局長：和田　洸記
  • 副局長：吉田　智紀
  • 副局長：山田　長宗

    装飾局

    　装飾局はブルゾンのデザインや、ステージパネル、エントランス、タイムテーブルの作成、設置といった、デザインやモノづくりの仕事がメインになります！ 現在、局員が情報の男3人しかいないので、デザインが得意な芸術や国際の人は大歓迎です！人数が少ない分、局員同士の距離が近いので先輩と仲良くなりたい人は是非、装飾局に！

    -------- 仕事内容 --------

  • ポスターコンクール
  • ブルゾンデザイン
  • ステージパネル制作(設営は他局と一緒に)
  • エントランス制作、設営

    • -------- メンバー --------

  • 局長：早川　優吾
  • 副局長：大脇　秀明

    広報局

    　各局のイラストを描かせていただきました～見てね！大学祭の宣伝を一緒に盛り上げてくれる方、 芸術学部の方でイラレの練習をしたい方(illustrator使えるよって子大募集です！)、優しい先輩たちと仲良くなりたい方にお勧めです！
    なんか難しそう、、、と思ったそこのあなた！私たちが一から丁寧に説明致します！今は3年生3人のみで活動しているので、1・2年生を大大大募集中！ 一緒に頑張る仲間が増えることを楽しみに待っています！

    -------- 仕事内容 --------

  • パンフレット作成
  • SNS運営
  • 校内案内用の標識作成
  • 当日の総合案内担当
